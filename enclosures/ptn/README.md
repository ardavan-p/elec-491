# elec-391/enclosures/ptn

# Enclosure for the PTN

This directory includes the baseplate, top cover, and various PCB models (nRF24L014 and PTN PCBs) for the PTN enclosure.

The entire assembly is provided as a pack and go zip file, named "Assembly_pack_and_go.zip".

The two assembly components are also available as STL files for ease of viewing - "baseplate.STL" and "top_cover.STL".
